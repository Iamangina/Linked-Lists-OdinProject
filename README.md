# Linked-Lists-OdinProject

A simple **singly linked list** implementation in JavaScript using a factory function.  
Supports basic operations like adding, removing, searching, and traversing nodes.

## Methods

- `append(value)` – adds a node to the end of the list  
- `prepend(value)` – adds a node to the start of the list  
- `size()` – returns the number of nodes  
- `getHead()` – returns the first node  
- `tail()` – returns the last node  
- `at(index)` – returns the node at a specific index  
- `pop()` – removes the last node  
- `contains(value)` – checks if a value exists (`true`/`false`)  
- `find(value)` – returns the index of a value or `null`  
- `toString()` – returns a string representation of the list: `(value) -> (value) -> null`  
- `insertAt(value, index)` – inserts a node at a specific index  
- `removeAt(index)` – removes a node at a specific index
